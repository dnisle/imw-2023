Ideas

At first, I was thinking about doing an interaction game and the user have to navigate to each sections using their keyboard in a pixel art style. But then my mom sent me a picture of my aunt's pond, back in my home town in Vietnam, called Ca Mau City, where she has a bunch of water plants and koi fishes in there. That's how I came up with this idea, since it's a bit easier and time save than the other one, I decided to go with this one.

Options

I came up with 3 options: a mini pond with floating lilypads and fishes; a half screen pond with floating lilypads and fished; and lastly lilypads that contain contents and floating around. All 3 supposed to have raining question mark and floating lilypads as a part of animation, and fished as interactive since the'd run a way when user tap them.

But after taking Peter's feedbacks into consideration, I decided to get rid of the questionmarks raining since it takes up to much page with little to no purpose, and went with full screen pond.

Coded canvas

Using html 5 canvas 5 I'm planning to make my interaction here. I choose this over p5 because I'm abit more familiar with this thanks to some classes in the pasrt

Initially, to create the ripple effect, I created on the position of mouseclicked a circle that increase it sizes. I then made it 3 different circles to make this effect better. Finally i deleted the circles when it goes out of view, to ensure the canvas does not lag.

I then created some new lilypads, I altered them a bit to make them look less like padman. I found some collision detection code on stack overflow for canvas and implement on my lilypads, making sure they bounce off eachother and the border (tho some of them are still buggy)

finally, i implemented small red dots as fishes, having them move to a random area in the canvas every few seconds. I made it so that when i clicked nera the fishes, this code would trigger also, making as if theyre running away from my tap.

And with the canvas is now out of the way (which took majority of my brain power before deciding), i just needed to resize and reposition the canvas to fit my website.
